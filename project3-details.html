Top 100 K-Drama Analysis

Project Overview: The K-Drama Analysis Dashboard is a sophisticated tool designed to provide in-depth insights into the performance and trends of Korean dramas (K-Dramas).The K-Drama Analysis Dashboard serves as an invaluable tool for stakeholders to gain a deep understanding of K-Drama trends and performance. By providing detailed insights and customizable reports, it aids in making data-driven decisions in the entertainment industry.
Technologies Used:
Frontend: HTML, CSS, JavaScript
Backend: IBM Cognos
