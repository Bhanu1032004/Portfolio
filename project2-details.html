Anti-Theft System For Vehicles

Project Overview:

I developed an advanced anti-theft system for vehicles, integrating robust UI/UX design using Figma for intuitive navigation and engaging visuals. The frontend utilized HTML, CSS, and JavaScript for real-time tracking and mapping, ensuring accurate and responsive location data. I enhanced security measures with a K-Nearest Neighbors (KNN) machine learning model, showcasing my proficiency in both frontend development and backend integration for comprehensive application functionality.

Frontend Tools:
UI/UX Design: Figma
Web Technologies: HTML, CSS, JavaScript
Mapping Integration: Real-time map integration using JavaScript libraries
Backend Tools:
Machine Learning: K-Nearest Neighbors (KNN) algorithm for predictive analysis
